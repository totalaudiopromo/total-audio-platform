# Total Audio Promo - Landing Page

Marketing landing page for [totalaudiopromo.com](https://totalaudiopromo.com)

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS
- TypeScript

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed via Vercel. Push to `main` triggers auto-deployment.
