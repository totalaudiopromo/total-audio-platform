## How to Install Skills

1. **Enable Skills in Claude Code:**
   - Settings → Capabilities → Enable "Skills"
   - Restart Claude Code

2. **Install via Plugin Marketplace:**

   ```
   /plugin → Add marketplace → Install skill collections
   ```

3. **Manual Installation:**
   - Download skill from GitHub
   - Unzip and place folder in `.claude/skills/`
   - Restart Claude Code
