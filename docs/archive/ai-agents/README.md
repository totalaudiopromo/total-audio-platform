## Project Context

**Total Audio Promo (TAP)**is an AI-powered music promotion platform serving independent artists and PR agencies. The platform combines smart automation with human expertise to deliver efficient, transparent music promotion services.

### Current Tech Stack

- **Backend**: Node.js/Express, Prisma ORM, PostgreSQL
- **Frontend**: Next.js with TypeScript, Tailwind CSS
- **Integrations**: Claude AI, Mailchimp, Airtable, Gmail APIs
- **Status**: MVP Ready with core features functional

### Key Goals

- No-BS approach to radio promo with guaranteed results
- Automation that feels human through AI-powered personalization
- Transparency over hype with real data and honest reporting
- DIY-friendly tools and pricing for independent artists
