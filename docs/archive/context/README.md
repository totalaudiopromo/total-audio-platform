# Development Context

This directory contains essential development context and standards files that provide guidance for AI assistants (Claude, Cursor, etc.) working on this project.

## Files Overview

### `cursor-context-master.md`
Master context file containing comprehensive project information, architecture details, and development guidelines for AI assistants.

### `cursor-component-standards.md`
Component development standards and best practices for building UI components in this project.

### `development-standards.md`
General development standards, coding conventions, and project-specific guidelines.

## Usage

These files should be referenced by AI assistants when:
- Understanding project architecture and requirements
- Writing or modifying code
- Following project-specific conventions
- Making architectural decisions

## For AI Assistants

When working on this project, please:
1. Review these context files to understand the project structure and standards
2. Follow the established patterns and conventions
3. Reference these files when making decisions about code structure or implementation
4. Update these files if you identify improvements or changes to the standards

## Project Context

This is the Total Audio Promo project - a comprehensive multi-tenant music promotion platform designed for both independent artists and PR agencies. The platform provides real-time campaign tracking, automated reporting, and seamless integrations with industry-standard tools.
